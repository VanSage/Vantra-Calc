# 🧮 Vantra Calc

A modern and advanced calculator built using **Python**, **Jupyter Notebook**, and **ipywidgets**. It provides multiple calculator modes, an interactive user interface, and supports a wide range of mathematical operations.

---

## ✨ Features

* Standard Calculator
* Scientific Calculator
* Programmer Calculator
* Unit Converter
* Calculation History
* Memory Functions (MC, MR, M+, M-)
* Scientific Functions
* Bitwise Operations
* Multiple Unit Conversions
* User-Friendly Interface

## 🚀 Getting Started

### Prerequisites

* Python 3.8 or later
* Jupyter Notebook

### Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Vantra-Calc.git
```

Move into the project folder:

```bash
cd Vantra-Calc
```

Install the required package:

```bash
pip install notebook ipywidgets
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```

Open **advanced_calculator.ipynb** and run all the cells.
